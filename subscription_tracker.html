📌 工具簡介

這是一個完全在本地瀏覽器運行的靜態網頁工具，專為記錄和計算各類電子媒體、串流服務及軟件的年費訂閱而設計。你只需輸入訂閱名稱和年費，月費及匯總將自動計算。

🔒 私隱與安全

✓ 完全離線所有資料只儲存在你自己的裝置上，絕不上傳至任何伺服器，亦不需要網絡連線運行。

✓ 無帳號登入無需註冊、無需電郵，沒有任何追蹤或分析代碼。

✓ 開放代碼此頁面為單一 HTML 檔案，你可以直接查閱所有源代碼，確保沒有隱藏功能。

💾 資料儲存方式

本工具使用瀏覽器內建的 LocalStorage 技術儲存資料。資料以 JSON 格式存放，key 名稱為 subscriptions。

  • 每次新增、修改、刪除或 Mute 操作後，資料會立即自動儲存
  • 關閉瀏覽器或重啟電腦後，資料仍然保留
  • 資料儲存在瀏覽器的本地沙盒內，其他網站無法讀取
  • LocalStorage 上限約為 5–10 MB，足夠儲存數千條訂閱記錄

🔐 持久化儲存（Storage Persist）

本工具啟動時會自動呼叫 navigator.storage.persist() 申請持久化儲存。取得授權後，即使裝置儲存空間不足，瀏覽器也不會自動清除本工具的資料。

  • 頁面右上方的「🔒 持久化」標記表示已成功申請
  • Chrome 通常會靜默授予；Firefox 可能彈出提示；Safari 支援有限
  • 即使未能取得持久化，資料仍受 LocalStorage 保護，只需避免手動清除瀏覽器資料

📤 匯入 / 📥 匯出備份

點擊「📥 匯出備份」按鈕，即可將所有記錄下載為 .json 檔案。點擊「📤 匯入備份」可從備份檔案還原資料，支援合併（保留現有資料並加入新記錄）或取代（清除現有資料後完整還原）兩種模式。

提示建議每隔一段時間匯出一次，存放在雲端硬碟（如 Google Drive、iCloud）以確保安全。

📲 加至主畫面（Add to Home Screen）

在 Android Chrome 瀏覽器開啟此頁面時，頂部會出現「加至主畫面」提示橫幅，點擊「安裝」即可將本工具安裝為 App 圖示，方便日後快速存取。iOS Safari 用戶可點擊分享按鈕，選擇「加至主畫面」手動安裝。

⚠️ 何時資料會消失？

注意以下情況會導致資料遺失，請定期使用「📥 匯出備份」功能：

  • 在瀏覽器設定中手動「清除瀏覽資料」並勾選了 Cookie / 本地儲存
  • 使用無痕模式（Incognito）開啟此頁面——無痕模式關閉後資料即消失
  • 更換瀏覽器或裝置（不同瀏覽器的 LocalStorage 互不相通）
  • 部分手機在儲存空間極度不足時，系統可能自動清除瀏覽器快取（已申請持久化則可避免）

🔇 Mute 功能說明

Mute 的用途是「暫時停用計算」，例如你已取消某訂閱但仍想保留記錄，或某費用為季節性支出，不想長期計入月費匯總。

  • Muted 項目會變灰並置於列表底部，不計入年費及月費總計
  • 再次點擊 🔊 按鈕即可恢復，資料不會遺失

🧮 計算方式

月費換算公式：月費 = 年費 ÷ 12（取小數點後一位顯示）

匯總只計算 未被 Mute 的項目，公式如下：

  • 年費總計 = 所有啟用項目年費之和
  • 月費總計 = 年費總計 ÷ 12
